# 📱 PhonePe Transaction Analytics Dashboard

An interactive **Power BI dashboard** designed to analyze PhonePe transaction data and uncover meaningful insights into payment performance, transaction trends, user behavior, and service utilization. This project demonstrates the use of **Power BI**, **Power Query**, **DAX**, and data visualization techniques to convert raw transactional data into actionable business insights.

---

## 📌 Project Overview

The rapid growth of digital payments has generated massive amounts of transaction data. This project analyzes PhonePe transaction records to understand transaction patterns, payment success rates, customer activity, and service usage through an interactive Power BI dashboard.

The dashboard enables stakeholders to monitor business performance using dynamic KPIs, charts, and slicers for better decision-making.

## 🎯 Business Objectives

The dashboard was developed to answer the following business questions:

- How many transactions were completed?
- What is the total transaction value?
- How many unique users use the platform?
- What is the overall payment success rate?
- How do transaction volumes change over time?
- Which services generate the highest transaction value?
- Who are the top users based on transaction value?
- What is the contribution of different age groups?
- How does weekday usage compare with weekend usage?
- How do payment statuses vary across different months?

---

## 📂 Dataset

The project uses a PhonePe transaction dataset containing transaction-level information, including:

- Transaction ID
- User ID
- Transaction Amount
- Service
- Service Type
- Payment Status
- Failure Reason
- Transaction Date

The dataset was cleaned, transformed, and modeled in Power BI before creating the dashboard.

**Dataset Included:** `Phonepe-Final-Dataset.xlsx`

## 📌 Key Insights

- Processed over **300,000 transactions** with a total transaction value of **₹3.47 Billion**.
- Achieved a **96% payment success rate**, indicating a highly reliable digital payment platform.
- Loan services generated the highest transaction value among all available services.
- Weekday transactions accounted for a significantly larger share of overall platform usage than weekends.
- Gen X and Millennials contributed the highest percentage of total transactions.
- Monthly transaction volume remained relatively stable throughout the year.
- Interactive filters allow users to analyze performance by month and payment status.
