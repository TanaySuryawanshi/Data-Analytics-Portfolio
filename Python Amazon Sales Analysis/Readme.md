# 🛍️ Amazon Sales Analysis using Python

A comprehensive **Exploratory Data Analysis (EDA)** project performed on Amazon sales data using **Python**. This project focuses on understanding sales trends, order fulfillment, customer distribution, product performance, and regional sales through data cleaning, visualization, and statistical analysis.

---

## 📌 Project Overview

The objective of this project is to analyze Amazon sales data and extract meaningful business insights using Python. The analysis includes data cleaning, preprocessing, exploratory data analysis, and visualization to identify sales patterns, customer behavior, and operational performance.

The project demonstrates the use of Python libraries for transforming raw data into actionable insights that can support business decision-making.

---

## 📂 Dataset

The dataset contains detailed information about Amazon orders, including:

- Order ID
- Order Date
- Order Status
- Product Category
- Product Size
- Quantity Ordered
- Sales Amount
- Fulfilment Method
- Sales Channel
- Courier Status
- Shipping City
- Shipping State
- Shipping Country
- B2B Orders

**Dataset Included:** `Amazon Sale Report.csv`

## 📊 Analysis Performed

The notebook explores the following business questions:

- Which product categories receive the highest number of orders?
- Which states contribute the highest sales?
- What is the distribution of product sizes?
- Which fulfilment method is used the most?
- What is the status distribution of customer orders?
- How are sales distributed geographically?
- What percentage of orders are B2B?
- Which courier statuses occur most frequently?

## 📈 Key Insights

- Identified the most popular product categories based on order volume.
- Analyzed geographical sales distribution across different states and cities.
- Evaluated order fulfilment performance and courier status.
- Compared B2B and B2C order distribution.
- Identified trends in product size preferences.
- Explored customer purchasing patterns using visual analytics.
