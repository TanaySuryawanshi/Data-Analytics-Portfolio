# 📂 Portfolio Projects

| Project | Description | Technologies |
|----------|-------------|--------------|
| 📈 **FNP Sales Analysis Dashboard** | Developed an interactive Excel dashboard to analyze sales performance, revenue trends, customer spending, product categories, and delivery metrics using Pivot Tables, Pivot Charts, slicers, and advanced Excel features. | Excel |
| 📱 **PhonePe Transaction Analytics Dashboard** | Built an interactive Power BI dashboard to analyze transaction trends, payment success rates, service performance, and user behavior using Power Query, DAX, and data modeling. | Power BI, DAX, Power Query |
| 🛒 **Customer Shopping Behavior Analytics** | Designed an end-to-end retail analytics solution involving Python preprocessing, PostgreSQL database management, SQL analysis, and Power BI dashboards to derive customer behavior insights. | Python, PostgreSQL, SQL, Power BI |
| 🛍️ **Amazon Sales Analysis using Python** | Performed exploratory data analysis on Amazon sales data to identify sales trends, customer behavior, product performance, and geographical insights using Python and visualization libraries. | Python, Pandas, NumPy, Matplotlib, Seaborn |
| 🛒 **Zepto SQL Data Analysis** | Conducted SQL-based business analysis on Zepto product inventory, including pricing strategy, discounts, category performance, and revenue estimation using PostgreSQL. | PostgreSQL, SQL |
