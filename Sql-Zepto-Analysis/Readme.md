# 🛒 Zepto SQL Data Analysis

A comprehensive SQL data analysis project on the **Zepto Product Dataset**. This project demonstrates data exploration, data cleaning, and business-oriented analysis using **PostgreSQL** to extract meaningful insights from an e-commerce inventory dataset.

---

## 📌 Project Overview

This project analyzes Zepto's product inventory to understand pricing strategies, discounts, stock availability, product categories, and inventory distribution.

Using SQL, the raw dataset was cleaned and transformed before performing analytical queries to answer real-world business questions.

---

## 🎯 Business Objectives

The analysis aims to answer the following questions:

- How many products are available in the dataset?
- Which product categories are available?
- How many products are in stock and out of stock?
- Which products have the highest discounts?
- Which high-priced products are currently unavailable?
- Which categories generate the highest estimated revenue?
- Which categories provide the highest average discounts?
- Which products offer the best value based on price per gram?
- How is inventory distributed across different weight categories?
- What is the total inventory weight available for each category?

---

## 📂 Dataset

The project uses a Zepto grocery products dataset containing product-level information such as:

- Product Name

- MRP

- Discounted Selling Price
- Available Quantity

## 🧹 Data Cleaning Performed

The following preprocessing steps were carried out before analysis:

- Created the database table
- Imported product dataset
- Checked for missing values
- Removed products with zero MRP
- Converted prices from **paise to rupees**
- Validated data consistency

### 🔹 Business Analysis

The project answers several business questions, including:

### 1. Top 10 Best Value Products
Identify products offering the highest discount percentages.

### 2. High MRP Products Out of Stock
Find premium products that are currently unavailable.

### 3. Estimated Revenue by Category
Calculate the potential revenue generated by each product category.

### 4. Premium Products with Low Discounts
Identify expensive products that receive minimal discounts.

### 5. Categories with Highest Average Discounts
Compare categories based on average discount percentages.

### 6. Price per Gram Analysis
Determine which products provide the best value for money.

### 7. Product Weight Segmentation
Classify products into:

- Low
- Medium
- Bulk

using SQL CASE statements.

### 8. Inventory Weight Analysis
Calculate the total inventory weight available for each category.
